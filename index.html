    • Graph for Programming Languages

    Python

    Python is renowned for its simplicity, readability, and versatility, making it a popular choice for a wide range of applications, from web development to data analysis.

    Java

    Java is widely used for its platform independence, robustness, and extensive ecosystem, making it a top choice for building enterprise-level applications and large-scale systems.

    HTML

    HTML forms the backbone of the web, providing the structure and semantics for building web pages and applications, making it essential for front-end development and creating interactive user experiences.





    Graph for Frameworks

    Bootstrap

    Bootstrap is a popular front-end framework known for its responsive design, pre-designed components, and ease of use, enabling developers to create visually appealing and mobile-friendly websites with minimal effort.

    React

    React is a powerful JavaScript library for building user interfaces, favored for its component-based architecture, virtual DOM, and efficient rendering, making it ideal for creating interactive and scalable web applications

    Node.js

    Node.js is a runtime environment for executing JavaScript code server-side, renowned for its non-blocking I/O model, event-driven architecture, and vast ecosystem of packages, enabling developers to build fast and scalable back-end applications and APIs.





    Graph for File Transfers utilities

    GitHub

    GitHub is a leading platform for version control and collaboration, facilitating code sharing, project management, and continuous integration workflows for software development teams worldwide.

    FileZilla

    FileZilla is a popular FTP client renowned for its simplicity, reliability, and cross-platform compatibility, providing users with an intuitive interface for transferring files securely between local and remote servers

    WinSCP

    WinSCP is a versatile SFTP and FTP client for Windows, known for its user-friendly interface, robust file transfer capabilities, and support for various encryption protocols, making it a preferred choice for secure file management and synchronization tasks.





    Graph for Browsers

    Chrome

    Chrome is a widely-used web browser developed by Google, known for its speed, simplicity, and seamless integration with Google services, offering users a fast and efficient browsing experience

    Edge

    Edge is a web browser developed by Microsoft, featuring a clean and intuitive interface, built-in security features, and integration with Microsoft services, providing users with a streamlined browsing experience optimized for Windows environments.

    Firefox

    Firefox is an open-source web browser known for its privacy features, customization options, and support for web standards, offering users a flexible and secure browsing experience tailored to their preferences.





    Graph for Operating Systems

    Windows

    Windows is a widely-used operating system developed by Microsoft, known for its user-friendly interface, extensive software compatibility, and widespread adoption in desktop computing environments.

    Linux

    Linux is a free and open-source operating system kernel that powers various distributions (distros), known for its stability, security, and flexibility, making it popular for servers, embedded systems, and developer environments.

    Mac

    Mac, short for macOS, is the operating system developed by Apple for its Macintosh computers, known for its sleek design, seamless integration with Apple ecosystem, and focus on user experience, offering users a powerful and intuitive computing environment.
